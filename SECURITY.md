```markdown
# Security Policy

Dev-Time-Capsule is committed to keeping our users and contributors safe. Please follow the guidelines below to report vulnerabilities.

## Reporting a Vulnerability

- Do **not** create a public issue for security concerns.
- Contact the maintainers privately: `security@dev-time-capsule.com`.
- Provide as much detail as possible (steps to reproduce, affected versions, screenshots if needed).

## Response

- We will acknowledge your report within 48 hours.
- We aim to fix critical issues as quickly as possible and release patched versions.

## Scope

- Vulnerabilities in the codebase, APIs, or third-party integrations.
- Security concerns regarding user data or system access.
```
